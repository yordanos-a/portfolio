Projects:

Health Informatics Platform (HIP)

The Health Informatics Platform is a tool we've built that is helpful to understand the impact of different insulin types and dosage ranges on managing diabetes. For our research, we extracted real-world diabetes patients' datasets from a publicly available repository. We implemented our data model and algorithm using Java programming language using an object-oriented approach. This project demonstrates the application of Computer Science in Healthcare. I had the privilege of presenting this work at both McMurry University and Abilene Christian University.
